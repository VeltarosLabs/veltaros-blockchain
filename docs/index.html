A Go-based blockchain network with PoW, wallets, transactions, HTTP API and P2P.

GitHub Docs

What is Veltaros?

Veltaros is a learning-first blockchain implementation that is evolving toward a full Bitcoin/Ethereum-like node stack: consensus, networking, wallet signing, mempool, mining, sync, and tooling.

Coin

Symbol: VLT

Purpose: fees + mining reward (network incentive)

Node

HTTP API for clients + P2P for syncing blocks and transactions.

Roadmap

Persistence → stronger tx model → difficulty/retarget → discovery → explorer.
